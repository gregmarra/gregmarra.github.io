About Spider Graphs for Google Docs

Spider Graphs for Google Docs allow you to create Spider Graphs for your data. Spider Graphs are a type of graph used to display the differences between several different sets of data across a range of categories. You may have seen Spider Graphs in racing games to compare car statistics, or in many other situations.

Use Spider Graphs for Google Docs

Using Spider Graphs for Google Docs is very simple. Just add the widget to your Google Docs spreadsheet, then select the cells that comprise the data you'd like to graph.
